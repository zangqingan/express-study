# 通用内容管理系统-开发指令

## 项目概述

本项目设计采用 **Vue 3 + TypeScript + Naive UI** 作为前端核心栈，前台借助 Nuxt 3 实现 SSR 满足 SEO，后台使用 Vite + SPA 保证开发体验。后端采用 **原生 Express** + TS + prisma+mysql，保持轻量、透明、高性能，工程化方面引入 Monorepo、自动化工具和测试体系，保障团队协作和长期维护。

## 开发规范

- 使用函数式组件 + Hooks
- 使用 Tailwind CSS 编写样式
- 组件要可复用
- 代码要有注释

## 设计要求

- 深色主题（背景 #0a0a0a，文字 #ffffff）
- 使用渐变色作为强调色
- 添加平滑的滚动动画
- 确保移动端适配

## 注意事项

- 保持设计简洁，不要过度设计
- 性能优化：图片使用懒加载
- 确保所有链接可点击
